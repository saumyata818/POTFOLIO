# Saumyata Nepal Portfolio

A simple terminal-style portfolio website inspired by a shell-themed portfolio interface.

## Files

- `index.html` — main page
- `styles.css` — visual styling and terminal theme
- `script.js` — interactive command handling

## Run locally

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

If you want to open the CV page directly, use the `cat cv` command in the terminal interface or open `cv.html` in your browser.
